drag the deej_run.exe into your startup folder.
to open the startup folder: 
	press win + r
	type shell:startup
	press enter
	drop the file in the folder

to edit config open config.yaml or rightclick on the deej icon in the system tray (while running)
to close programm rightclick on the icon in the system tray
